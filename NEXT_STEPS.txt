GIAI ĐOẠN 2 – LẮP RÁP PHẦN CỨNG (KHI CÓ ĐỒ)
──────────────────────────────────────────────────────────────────

[ ] 2.1. Phần cứng cần chuẩn bị
    - Raspberry Pi 5 (4GB hoặc 8GB RAM)
    - Flight Controller: Pixhawk 6C hoặc Cube Orange (ArduPilot)
    - ESC + Motor (4 cái) + Frame F450 hoặc tương đương
    - Camera: Raspberry Pi Camera Module 3 hoặc USB Camera
    - Microphone: USB Mic hoặc Mini Lavalier Mic (có foam chống gió)
    - Pin LiPo 4S + Sạc Pin
    - Còi báo động (Buzzer) + LED chỉ thị

[ ] 2.2. Cài đặt Raspberry Pi 5
    - Flash Raspberry Pi OS (64-bit) vào thẻ nhớ
    - pip install dronekit pymavlink websockets sounddevice
              numpy webrtcvad scipy opencv-python ultralytics
    - Copy scripts/drone_edge_client.py lên Pi
    - Cấu hình Pi Camera: sudo raspi-config > Interface > Camera

[ ] 2.3. Kết nối Pixhawk ↔ Raspberry Pi 5
    - Nối UART Telemetry2 của Pixhawk vào GPIO UART của Pi
    - Cấu hình MAVLink: --connect /dev/ttyAMA0 --baud 57600
    - Test kết nối: python -c "from dronekit import connect; v=connect('/dev/ttyAMA0')"

[ ] 2.4. Cân chỉnh Drone (Calibration)
    - Calibrate Compass, Accelerometer qua Mission Planner
    - Calibrate Radio (nếu dùng RC Controller dự phòng)
    - Chạy thử ở chế độ STABILIZE trước khi bật tính năng AI
